FAQ
===

Why DEAP?
---------

DEAP is a very flexible framework to run evoultionary algorithms. We can fine tune all aspects of the ``evolution`` and ``individuals``.

Why graph-tool?
--------------------------------------------------

graph-tool_ is an excellent python package which allows scripted analysis `Graph filtering <https://graph-tool.skewed.de/static/doc/quickstart.html#graph-filtering>`_ and `GraphViews <https://graph-tool.skewed.de/static/doc/quickstart.html#graph-views>`_ that are ideal for manipulating and visualising graph objects.

.. graph-tool: https://graph-tool.skewed.de/
